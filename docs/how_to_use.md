# How to Reuse This System for Any Brand

## Step 1
Open prompts/master_prompt.txt

## Step 2
Replace these fields:
- Product name
- Brand name
- Price
- Target audience
- Platform
- Tone

## Step 3
Paste into Claude (claude.ai) or ChatGPT

## Step 4
Save the output into the /outputs folder

## Step 5
Use content_pack/full_content_pack.md as your
client-ready deliverable

---

## Works For:
- Local salons, cafes, gyms
- D2C skincare, fitness, food brands
- SaaS tools
- Creator brands

## Tip:
Run the hooks prompt separately for 10 fresh hooks
every time you start a new brand campaign.
