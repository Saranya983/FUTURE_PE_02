# Prompt Logic — Why Each Prompt Is Structured This Way

## 1. Master Prompt
- Brand brief section ensures Claude understands context before writing
- "RULES" section prevents salesy language
- Asking for multiple formats in one prompt saves time

## 2. Hooks Prompt
- Categorized by hook style so output is varied
- 10 hooks gives enough options to A/B test
- Under 15 words rule matches real Instagram behavior

## 3. Platform Adaptation Prompt
- Each platform has different viewer behavior
- Facebook viewers read more → longer scripts
- Shorts viewers decide in 2 seconds → hooks first

## 4. CTA Prompt
- Avoiding "Buy Now" increases trust in UGC content
- Friend-recommendation tone converts better in D2C
- Urgency CTAs and curiosity CTAs serve different audiences
