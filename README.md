# UGC Ad Content Pack — AI-Powered Marketing

## 📌 Product Chosen
**GlowLab Vitamin C Face Serum**
A D2C skincare brand selling online across India.

## 🎯 Problem This Ad Solves
Many women struggle with dull skin, dark spots, and uneven skin tone.
They spend money on expensive products that don't work.
This UGC ad positions GlowLab as the honest, affordable solution.

## 🤖 Tools Used
- Claude (claude.ai) — for scripts and tone matching
- ChatGPT — for rapid hook generation
- GitHub — for documentation

## 📁 Folder Structure
- /prompts → All reusable prompt templates
- /outputs → All AI-generated content
- /content_pack → Final combined deliverable
- /docs → Prompt logic and usage guide

## 🔁 How to Reuse This for Any Brand
1. Open prompts/master_prompt.txt
2. Replace product name, audience, and tone
3. Paste into Claude or ChatGPT
4. Save outputs in /outputs folder
